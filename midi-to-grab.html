Tutorial
To export this as a level, import your midi, press "Build level" then Download the Json. After downloading the Json, go to grabvr.tools and press file > insert then click .JSON File, afterwards press File > Export > Export the .level, then paste the level file into your grab file using Cx file explorer, ZArchiver, etc.
